# 🌍 TripAdvisor Clone

A responsive and visually appealing TripAdvisor clone website built using HTML and CSS. This project recreates the user interface of the popular travel platform, allowing users to explore destinations, discover experiences, and browse travel-related content.

## ✨ Features

- Responsive and modern user interface
- Attractive hero section
- Destination and travel cards
- Search and navigation bar
- Recommended places and experiences
- Interactive hover effects and animations
- Clean and organized design

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Responsive Design# Trip-Advisor-clone-
